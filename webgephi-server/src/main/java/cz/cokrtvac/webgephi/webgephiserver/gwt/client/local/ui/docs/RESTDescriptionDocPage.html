REST services description

All resources (URLs) are relative to {server_base}/rest/. E. g. https://www.webgephi.cz/rest/

Error response

If something is wrong, error response is returned instead of regular xml or html (with corresponding http status).
XML error message 
<?xml version="1.0" encoding="UTF-8"?>
<error>
    <code number="409">Conflict</code>
    <message>User is invalid</message>
    <detail>Username test is already taken.</detail>
</error>
HTML error message 
<?xml version="1.0" encoding="UTF-8"?>
<html>
<head>
    <title>Error 404 (Not Found)</title>
</head>
<body>
    <h1>404 (Not Found)</h1>
    <ul>
        <li>Message: This graph has no statistics</li>
        <li>Detail: Statistics are set only if some statistic function was applied previously</li>
    </ul>
</body>
</html>
