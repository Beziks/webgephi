Welcome to Webgephi SERVER


About this application

This application is a web interface over an opensource application Gephi (Gephi toolkit to be exact). Gephi is a JavaSE (desktop) application for graph manipulation and visualization.

Gephi is an interactive visualization and exploration platform for all kinds of networks and complex systems, dynamic and hierarchical graphs.

Webgephi provides a REST interface to most of Gephi functionality. This application served only as a backend, for an end user, there is a frontend application Webgepi client (or anybody can write his own frontend app).

For regular users...

You was probably redirected here from some client application. If so, login (or create an account here) and provide access to your account to the client application.

Anytime, you can use an official client application to manipulate your graphs.

For developers...

This application provides access to Gephi functionality using REST interface. If you want to use it in your project, create an account here and register your application.

Access to accounts of other users is provided using OAuth1 (recommended solution) or Basic authentication (discouraged). REST interface is available on /rest path.

For more information, see documentation of REST interface and documentation about authorization. All project is available on Github.
